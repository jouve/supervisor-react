supervisor-react
================

A web interface for supervisor made with React.

```
usage: supervisor-react [-h] [-d] [-p PORT] [-s SUPERVISOR] [-v]

optional arguments:
  -h, --help            show this help message and exit
  -d, --debug
  -p PORT, --port PORT
  -s SUPERVISOR, --supervisor SUPERVISOR
  -v, --verbose
```
